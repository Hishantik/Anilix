# Disclaimer

## About Anilix

Anilix is a terminal-based tool designed to simplify how users discover and watch anime online. It acts as a lightweight, purpose-built interface — much like a web browser, but stripped down to only what's needed for finding and playing anime streams.

All anime content accessed through Anilix is fetched from third-party sources that are publicly available on the internet. Anilix does not host, store, or redistribute any media. It merely automates the process of locating and requesting content that these sources already make available to any standard browser.

If you operate a source site referenced by this project, the methods used are transparent and visible in the source code. You are encouraged to implement appropriate protections on your end if you wish to restrict automated access.

## DMCA and Copyright

Like a web browser, Anilix is a neutral tool. It does not select, curate, or promote any specific content. What a user chooses to access with it is entirely their own decision.

Anilix operates using client-side requests — it does not bypass DRM, circumvent access controls, or exploit server vulnerabilities. It simply requests publicly available resources in an automated fashion.

If you are a copyright holder and believe your content is being served without authorization through a source that Anilix connects to, your claim should be directed at the hosting provider — not at this project. Anilix does not host or serve any content itself.

As of the time of writing, streams are resolved through third-party hosting services that Anilix has no affiliation with. All DMCA and takedown requests should be directed to the relevant hosting provider.
